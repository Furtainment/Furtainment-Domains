## **About This Document**

Not all domains or subdomains are currently active or have a live website.  
Inactive entries are included to indicate **planned rollouts**, **reserved namespaces**, or **future product lines** within the Furtainment ecosystem.

This document exists to:

- Ensure future **third‑party scripts**, **programs**, and **apps** can function correctly  
- Provide **transparency and trust** for developers working with Furtainment domains  
- Document **redirects** so mistyped traffic continues to flow to the correct domain extension  
- Support long‑term **ecosystem stability**, **SEO clarity**, and **infrastructure planning**

## **Repository Structure**

Repository folders may contain:

- **Scripts** — utilities for domain handling, redirects, or automation  
- **Datasets** — structured domain metadata  
- **Developer Resources** — documentation, examples, and integration notes  
- **Future Tools** — planned utilities for app and program developers  

These resources exist to support developers creating apps, scripts, and programs that interact with the Furtainment network.
